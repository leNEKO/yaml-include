# List of examples

You can find in this folder some examples of how you can use this bin/lib

The output is located in `examples/_expected_outputs`

Here the list of examples:
* `examples/sample`: Simple example with different kind of files
* `examples/deep_include`: Show example with `!include` with safe circular recursion
* `examples/circular_error`: Minimal example that will fails if used with 'error_on_circular' param
* `examples/glob`: Minimal example of glob include
* `examples/glob_ignored_local_yaml`: Show how glob ignore same level circular include for easier folder structure
